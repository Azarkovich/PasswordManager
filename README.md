# --------------------- PasswordManager - Projet Évolutif --------------------

Ce dépôt regroupe l'évolution (sur 3 versions) de mon gestionnaire de mots de passe.

| Version |    Nom du dossier    | Description                       |
|---------|----------------------|-----------------------------------|
| **1.0** | `PasswordManager1.0` | CLI basique avec stockage local | Python (CLI)
| **2.0** | `PasswordManager2.0` | Interface Tkinter / SQLite | Python
| **3.0** | `PasswordManager3.0` | UI Qt6, chiffrement AES-GCM, modèle zéro-knowledge | PySide6

## ---------- Version actuelle
-> La version courante(3.0 -- PassMan) est la pus complète à ce jour et continue d'évoluer.

## ---------- Fonctionalités 
- Coffre chiffré AES-GCM
- Zero Knowledge Encryption
- Import/Export chiffré
- Générateur de mots de passe + ajout direct au coffre
- Gestion des entrées (édition, tri, recherche)
- Interface Qt native (supportable Windows/Linux/macOS)
